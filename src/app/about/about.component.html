This Angular project simulates two CISCO switches.

Features

  • It models the Interconnecting Cisco Networking Devices, Part 2 (ICND2) exam .
  • It runs on Node version 19.8.0.
  • It uses Angular CLI version 1.0.0.
  • It uses Angular version 4.4.7
  • It uses Typescript version 4.9.5.
  • The mustache@2.3.2 NPM library is used to render text templates with placeholders.
    • The placeholders are replaced with values from the device model
    • The device model is updated as commands are performed on each switch
  • The multi-integer-range@4.0.0 NPM library is used to take a CISCO interface range and convert it to a numeric array.
  • The event-dispatch@0.4.1 NPM library is used to dispatch events.
  • The class-transformer@0.5.1 library is used to convert literal objects to class objects.
